Lista Ordenada

Para criar uma lista ordenada, usamos a Tag ol e a Tag li para declarar os elementos da lista.

Exemplo:

  1. Café
  2. Suco
  3. Leite

Lista não Ordenada

Para criar uma lista não ordenada, usamos a Tag ul e a Tag li para declarar os elementos da lista.

  • Café
  • Suco
  • Leite
