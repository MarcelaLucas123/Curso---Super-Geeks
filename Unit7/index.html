Framework Bootstrap

O que é um framework?

Um framework é um conjunto de ferramentas, bibliotecas e padrões que fornecem uma estrutura predefinida para o desenvolvimento de software.

O que é o Bootstrap?

O bootstrap é um framework front-end open-source usado para facilitar o desenvolvimento de sites e aplicações web responsivas.
